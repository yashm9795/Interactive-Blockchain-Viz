What is BlockChain?

     

What is a block ?

A block is like a page or a ledger that consist of mainly three components .

It consists of the data, a hash to protect the data and a previous hash inorder to form a chain of blocks.In the world of cryptocurrency, a block contains more than 500 transactions on average. The average size of a block seems to be 1MB. In Bitcoin Cash ( a hard fork from the Bitcoin blockchain ), the size of a block can go up to 8MB. This enables more transactions to be processed per second.

 

How is a Block created ?

As we mentioned earlier, a block consists of data. This data could be transactions between users of the block-chain. To understand how a block is created let's imagine that we are miners

Transactions taking place in the network are sent to all nodes. Since we are miners we receive these transactions as well. Click "Receive New Transaction" to see a new Transaction added. The limit on number of transactions we will be adding to our block is 5

Now that we have all our transactions. Click on them to approve them except the last one and then click "Attempt Block Creation". Did it fail ? This is because all the transactions in a Block need to be approved and validated by the miner. This ensures valid transactions in the block-chain. Now approve all the last transaction too and click the button. You'll see that a New Block is created

How is a Block approved to get added to the Chain ?

Miner ! You have now validated transactions and created a block. In the background you solved a complex mathematical problem, the goal of which was to find a number known as the "nonce". This was used to create a hash for the block you created. Now we want to verify if the solution to this mathematical problem was correct or not. So we need to ask our fellow Miners.

 

Try clicking on "Add Block" in the Next Section. If your block didn't get approved by peers you would not be able to add this block to the Blockchain. You then have to start creating the block all over. If your block got appoved, click the "Add Block" button to add your newly created block to the chain

*Only once your block is approved by peers, can it get added to chain*

How is a Block-Chain created from a Block ?

Once you have the data you want to include in a block, the SHA-256 algorithm generates a new hash that has to be used for this block, the block then utilizes the hash for the previous block in order to create the chaining effect.

You can view how this works in the visualization. Click on add block to add a new block. Notice how the new block includes the previous block's hash in it. This chaining is what ensures that the blocks have a sequence to them and also ensures security. For adding your own block, you would have to correct subsequent blocks and therefore making changes to block is extremely difficult in blockchain
