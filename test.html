What is BlockChain?

     

What is a block ?

A block is like a page or a ledger that consist of mainly three components .

It consists of the data, a hash to protect the data and a previous hash inorder to form a chain of blocks.

 

How is a Block-Chain created from a Block ?

Once you have the data you want to include in a block, the SHA-256 algorithm generates a new hash that has to be used for this block, the block then utilizes the hash for the previous block in order to create the chaining effect.

You can view how this works in the visualization. Click on add block to add a new block. Notice how the new block includes the previous block's hash in it. This chaining is what ensures that the blocks have a sequence to them and also ensures security. For adding your own block, you would have to correct subsequent blocks and therefore making changes to block is extremely difficult in blockchain

How is a Block approved to get added to the Chain ?

Once a miner has verified the transactions that he listens to on the node. A decision is made on which transactions to include in the block. After this, the miner solves a complex mathematical problem, the goal of which is to find a number known as the "nonce"
